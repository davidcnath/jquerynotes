Document Ready Function

We always want to put our jQuery inside a .ready function so that the code does not begin until the browser has loaded all of the html data. Otherwise any script may not work properly. The code looks as follows:

  • $(document).ready(function(){
  • // code goes here
  • });

Selecting Elements

Below is an unordered list with id="mylist" and each list item has class="item".

  • One
  • Two
  • Three
  • <ul id="mylist">
  • <li class="item">One</li>
  • <li class="item">Two</li>
  • <li class="item">Three</li>
  • </ul>

To call an element we need to use a selector:

  • $('li') would select all elements in the page witch in this case are list items.
  • $('.item') would select all elements with the class 'item'.
  • $('#mylist') would select any element with the id 'mylist'.
  • $('#mylist li') would select all the <li> elements which are under the element with id 'mylist'.
  • $('#mylist > li') using a greater than sign however would only select the <li> elements which are direct children of the element with id 'mylist'.
  • $('#mylist, .item') with a comma to seperate you can select multiple elements.
  • $('#mylist, li:first') this selects the first list item under the element with id 'mylist'. You can also replace 'first' with 'last', 'even' or 'odd'.

Traversing the DOM (Document Object Model)

There are 'shortcuts' to navigating your way around HTML elements.

  • $('#mylist').find('li'); will return all <li> elements within the #mylist parent element.
  • $('li').last(); will return the last <li> element. Likewise, .first() would return the first.
  • $('li').first().next().prev(); Methods can be strung together to traverse up and down the DOM.
  • $('li').first().parent(); This will return the parent, in this case the <li> with id 'mylist'.
  • $('#mylist').children('li'); Similar to find() but this will only return the <li> elements that are children of 'mylist'.
  • $('#mylist').closest('li'); This returns the closest <li> element.

Creating an Element

For example, an element we might want to create is a list item in order to add it to a list. This may look something like: var element = $('<li>Don't forget the carrots</li>');.

Adding and Removing Elements

If we wanted to add an element, such as a new item to our list we can do that using the following functions:

  • .append(<element>) adds an element to the end of the selected elements children.
  • .prepend(<element>) adds an element to the beginning of the selected elements children.
  • .after(<element>) adds a new element after the selected element.
  • .before(<element>) adds a new element before the selected element.
  • .remove(<element>) removes the selected element.

For example: $('#mylist').prepend(element);

There are also four very similar ways to append to the DOM. These do the same thing but is written differently

  • .appendTo(<element>)
  • .prependTo(<element>)
  • .insertAfter(<element>)
  • .insertBefore(<element>)

For example: element.appendTo($('#mylist'));

Changing an Element's Text or HTML

Using .text() or .html() we can both get and edit code within an element. The two are very similar but have their differences - the clue is in the name.

  • $('<element>').text();
  • $('<element>').text("Enter text here");

The first line would return the text from the selected element, whereas the second line would replace any text within the element.

Interactions with .on()

We use the following structure to interact with elements on the page.

  • $('<element>').on('action',function(){
  • // code goes here
  • });

Common elements to see here would be things like buttons and the on-action likely to be on 'click'. Then pretty straightforward, next a function is called. Other actions can be 'mouseenter' and 'mouseleave'.

Changing the .css()

The code below is an example of a button interaction to change the css background-color of the square box. The buttons each have id's of '#green' and '#orange' and the box '#mybox'.

  • $('#green').on('click',function(){
  • $('#mybox').css("background-color","green");
  • });
  • $('#orange').on('click',function(){
  • $('#mybox').css("background-color","orange");
  • });

This Keyword

This is an important keyword as it refers back to the element we have selected. For example, if we had a button with the code:

  • $('button').on('click', function(){
  • $(this).remove();
  • });

this would be referring to the originally selected button, and hence remove itself. Below is an example of using this to change the property of a box when you move the mouse over it, with the following code.

  • $('#greybox').on('mouseenter', function(){
  • $(this).addClass('longbox').removeClass('shortbox');
  • });
  • $('#greybox').on('mouseleave', function(){
  • $(this).addClass('shortbox').removeClass('longbox');
  • });

.on() also has a Selector

Before the function and after the action, we can add a selector to the .on() method. In this example, if we had a list of different elements on a page, the following code would only select the button elements within that list. The same would go for classes, if say you wanted to change the style of every list element of a certain class.

  • $('#mylist').on('click', 'button', function(){
  • // code goes here
  • });

.filter()

Similarly when calling an element you can filter out those elements or classes that you want additionally. For example the following code:

  • $('.under50pounds').filter('.onsale')

To which we may choose to add a styling class for any items that are under 50 pounds and also on sale.

Hidden Elements and Toggle

As well as changing the style of elements or adding CSS and classes, we can show or hide very easily with jQuery by adding the CSS property "display: none;". Here is an example with the javascript code and several types of toggle: '.toggle()', '.slideToggle()' and '.fadeToggle()'.

  • $('#togglebox').on('click',function(){
  • $('#hiddenbox').toggle();
  • });


Keyboard Events (immediate feedback)

We can have jQuery recieve data from user input and return it after performing some kind of action on it. For example having the user input values and return the sum of those values, as shown below. Here we use 'keyup' instead of 'onclick' and in order to get the value we use '.val()'.

  • $('#mymathlist').on('keyup', function(){
  • var numone = +$('#numone').val();
  • var numtwo = +$('#numtwo').val();
  • $('#myanswer').val(numone + numtwo);
  • });
  • + =

It is also important that we include a '+' symbol before we collect the value of the input as that ensures only numbers are received as opposed to strings.

Animating with jQuery

  • } else {
  • $(this).animate({"left":"0px"},'slow');

Here we target the '#smallbox' id and do our usual on.('click',function(){... The method used to move the box is .animate() wherein you enter CSS properties and optionally how fast you want the animation to happen.
